````{admonition} Übung 7.7 (✩✩✩)
Es gelten folgende Regeln:

Simulieren Sie 5000 Testdurchläufe und ermitteln Sie:

2. Wie viele Durchläufe lösten einen Alarm aus (mindestens ein Fehler)?
3. Wie viele Durchläufe verliefen ohne Ereignis (kein Fehler)?

Geben Sie die absoluten Häufigkeiten und die prozentualen Anteile mit
````

```{code-cell} python
# Code-Zelle
```

````{admonition} Lösung
:class: tip
:class: dropdown
```python
import numpy as np
import plotly.express as px

# Eingabe
ANZAHL_TESTS = 5000

# Verarbeitung
anzahl_abschaltung = 0   # beide Sensoren Fehler
anzahl_alarm = 0         # genau ein Sensor Fehler
anzahl_ok = 0            # kein Fehler

for i in range(ANZAHL_TESTS):
    # Zufallszahl 1 bis 10, Fehler bei Ergebnis 1 (10 % Wahrscheinlichkeit)
    sensor_1 = np.random.randint(1, 11)
    sensor_2 = np.random.randint(1, 11)

    fehler_1 = (sensor_1 == 1)
    fehler_2 = (sensor_2 == 1)

    if fehler_1 and fehler_2:
        anzahl_abschaltung = anzahl_abschaltung + 1
    elif fehler_1 or fehler_2:
        anzahl_alarm = anzahl_alarm + 1
    else:
        anzahl_ok = anzahl_ok + 1

# Prozentualer Anteil
anteil_abschaltung = anzahl_abschaltung / ANZAHL_TESTS * 100
anteil_alarm = anzahl_alarm / ANZAHL_TESTS * 100
anteil_ok = anzahl_ok / ANZAHL_TESTS * 100

# Ausgabe
print(f"{'Ereignis':<20} {'Anzahl':>8} {'Anteil':>8}")
print(f"{'-'*38}")
print(f"{'Abschaltung':<20} {anzahl_abschaltung:>8d} {anteil_abschaltung:>7.1f} %")
print(f"{'Alarm':<20} {anzahl_alarm:>8d} {anteil_alarm:>7.1f} %")
print(f"{'Kein Ereignis':<20} {anzahl_ok:>8d} {anteil_ok:>7.1f} %")

fig = px.bar(x=["Abschaltung", "Alarm", "Kein Ereignis"],
             y=[anzahl_abschaltung, anzahl_alarm, anzahl_ok],
             labels={"x": "Ereignis", "y": "Anzahl Testdurchläufe"},
             title=f"Sensorsimulation: {ANZAHL_TESTS} Testdurchläufe")
fig.show()
```

Theoretische Erwartungswerte bei einer Fehlerwahrscheinlichkeit von je 10 %:
* Abschaltung (beide Fehler): 10 % × 10 % = 1 %
* Alarm (genau ein Fehler): 2 × 10 % × 90 % = 18 %
* Kein Ereignis: 90 % × 90 % = 81 %

Die Kombination aus `and` (Abschaltung) und `elif`-`or` (Alarm) sorgt dafür,
dass jeder Testdurchlauf genau einem der drei Fälle zugeordnet wird.
````

````{admonition} Lösung
    if (gewicht_kg < 2.0) and (seite_cm < 20):
        anzahl_band1 = anzahl_band1 + 1
    elif (gewicht_kg > 10.0) or (seite_cm > 60):
        anzahl_band3 = anzahl_band3 + 1
    else:
        anzahl_band2 = anzahl_band2 + 1

# Ausgabe
print(f"Band 1 (Kleinsendung):   {anzahl_band1:4d} Pakete  ({anzahl_band1/ANZAHL_PAKETE*100:.1f} %)")
print(f"Band 2 (Standardpaket):  {anzahl_band2:4d} Pakete  ({anzahl_band2/ANZAHL_PAKETE*100:.1f} %)")
print(f"Band 3 (Sperrgut):       {anzahl_band3:4d} Pakete  ({anzahl_band3/ANZAHL_PAKETE*100:.1f} %)")

fig = px.bar(x=["Band 1\n(Kleinsendung)", "Band 2\n(Standardpaket)", "Band 3\n(Sperrgut)"],
             y=[anzahl_band1, anzahl_band2, anzahl_band3],
             labels={"x": "Förderband", "y": "Anzahl Pakete"},

fig.show()
````
