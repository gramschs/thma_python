bauteile.remove('Schraube M8')
print(bauteile)

# Nochmaliges Entfernen
bauteile.remove('Schraube M8')

# Gegeben: Zwei separate Einkaufslisten
einkauf_alice = ['Chips', 'Cola', 'Salsa', 'Nachos']

einkaufsliste = einkauf_alice + einkauf_bob

- `.append()` zum Hinzufügen neuer Elemente
- `.remove()` zum Löschen unerwünschter Elemente  
- Schreibzugriff mit Index zum Ersetzen von Elementen
- `len()` zur Bestimmung der finalen Anzahl

Wichtig: Bei Schritt 4 muss man wissen, dass 'Cola' an Index 1 steht (zweite Position), um es durch 'Wasser' zu ersetzen.

Erstellen Sie ein Liniendiagramm mit Plotly Express, das folgende Anforderungen erfüllt:

2. Erstellen Sie einen Lineplot mit `px.line()`, wobei `kraft_N` die x-Achse und `dehnung_mm` die y-Achse darstellt.

:class: tip
:class: dropdown

- `.show()` erzwingt die Anzeige des Diagramms

messungen = []

messungen.append(temp1)
