Übung 2.1

Betrachten Sie die folgenden beiden Code-Zeilen:

print(5 + 3)
print('5' + '3')

  1. Sagen Sie vorher, was jeweils ausgegeben wird.

  2. Führen Sie den Code in der nächsten Code-Zelle aus und überprüfen Sie Ihre Vorhersage.

  3. Erklären Sie, warum die Ausgaben unterschiedlich sind.

# Code-Zelle
Lösung

  1. Vorhersage

  • print(5 + 3) gibt 8 aus

  • print('5' + '3') gibt 53 aus

  1. Ausführung Beim Ausführen des Codes sehen Sie genau diese Ausgaben.

  2. Erklärung

  • In der ersten Zeile werden zwei Integer (Ganzzahlen) addiert. Der +-Operator führt eine mathematische Addition durch: 5 + 3 = 8

  • In der zweiten Zeile werden zwei Strings (Zeichenketten) verkettet. Die Anführungszeichen machen aus den Zahlen Texte. Der +-Operator hängt bei Strings die Zeichenketten hintereinander: ‘5’ + ‘3’ = ‘53’

Wichtige Erkenntnis: Der Datentyp bestimmt, wie Operationen durchgeführt werden. Derselbe Operator (+) verhält sich bei Zahlen und Strings völlig unterschiedlich!

Übung 2.2

Ein Stahlträger hat eine Länge von 3.5 m und wiegt pro Meter 25.4 kg.

  1. Speichern Sie die Länge in einer Variable laenge_m und das Gewicht pro Meter in einer Variable gewicht_pro_meter.

  2. Berechnen Sie das Gesamtgewicht des Trägers mit der Formel: Gesamtgewicht = Länge · Gewicht pro Meter.

  3. Geben Sie das Ergebnis mit print() aus.

# Code-Zelle
Lösung

gewicht_pro_meter = 25.4

# Berechnung

Übung 2.7

Gegeben ist folgender Code:

spannung_V = 230

print(spannung_V)

  1. Führen Sie den Code gedanklich Schritt für Schritt aus und notieren Sie nach jeder Zeile den aktuellen Wert der Variable spannung_V.

  2. Erstellen Sie eine Tabelle mit zwei Spalten: “nach Zeile” und “Wert von spannung”

  3. Führen Sie den Code anschließend in der Code-Zelle aus und überprüfen Sie Ihre Vorhersage.

Hinweis: Denken Sie daran, dass der Zuweisungsoperator = in zwei Schritten arbeitet: Erst wird die rechte Seite berechnet, dann wird das Ergebnis der Variable auf der linken Seite zugewiesen.

# Code-Zelle
Lösung

Schrittweise Ausführung:

nach ZeileWert von spannungBerechnung
spannung_V = 230230Initialisierung
spannung_V = spannung_V * 2460230 * 2 = 460
spannung_V = spannung_V - 100360460 - 100 = 360

  4. Korrigieren Sie den Code, sodass er korrekt funktioniert.

Tipp: Es gibt zwei Stellen, an denen Typkonvertierungen fehlen!

# Code-Zelle für korrigierten Code

Übung 10 (Mini-Projekt)

Sie sollen einen Rechner entwickeln, der für ein quaderförmiges Stahlbauteil das Volumen, das Gewicht und die Materialkosten berechnet.

Benötigte Formeln:

  • Volumen: V = Länge · Breite · Höhe (in m³)

  • Gewicht: m = V · ρ (in kg), wobei ρ die Dichte ist

  • Kosten: K = m · Preis (in €)

Ihr Programm soll

  1. nach folgenden Werten fragen:

    • Länge des Bauteils in Metern,

    • Breite des Bauteils in Metern,

    • Höhe des Bauteils in Metern,

    • Dichte von Stahl in kg/m³ (typisch: 7850 kg/m³),

    • Preis pro Kilogramm in € (typisch: 2-5 €/kg).

  2. Das Volumen, das Gewicht und die Gesamtkosten berechnen.

  3. Alle Ergebnisse übersichtlich ausgeben:

    • “Volumen: {Wert} m^3”

    • “Gewicht: {Wert} kg”

    • “Materialkosten: {Wert} EUR”

Anforderungen:

  • Verwenden Sie aussagekräftige Variablennamen.

  • Strukturieren Sie Ihren Code mit Kommentaren nach dem EVA-Prinzip.

  • Verwenden Sie Zwischenvariablen für die Berechnungsergebnisse.

  • Achten Sie auf korrekte Typkonvertierungen.

Testen Sie Ihr Programm mit einem Beispiel: Länge = 2.0 m, Breite = 0.5 m, Höhe = 0.3 m, Dichte = 7850 kg/m³, Preis = 3.50 €/kg.

# Code-Zelle
Lösung
# Eingabe
laenge_m = float(input('Geben Sie die Länge des Bauteils in Metern ein: '))
breite_m = float(input('Geben Sie die Breite des Bauteils in Metern ein: '))
hoehe_m = float(input('Geben Sie die Höhe des Bauteils in Metern ein: '))
dichte_stahl_kg_m3 = float(input('Geben Sie die Dichte von Stahl in kg/m³ ein: '))
preis_pro_kg = float(input('Geben Sie den Preis pro Kilogramm in € ein: '))

# Verarbeitung
volumen_m3 = laenge_m * breite_m * hoehe_m
gewicht_kg = volumen_m3 * dichte_stahl_kg_m3
materialkosten_euro = gewicht_kg * preis_pro_kg

# Ausgabe
print('Volumen: ' + str(volumen_m3) + ' m^3')
print('Gewicht: ' + str(gewicht_kg) + ' kg')
print('Materialkosten: ' + str(materialkosten_euro) + ' EUR')

Beispiel-Ausgabe (mit den Testwerten: 2.0, 0.5, 0.3, 7850, 3.50):

Volumen: 0.3 m^3
Gewicht: 2355.0 kg
Materialkosten: 8242.5 EUR
