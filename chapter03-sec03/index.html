Übung 3.1

Erzeugen Sie die folgenden Listen und geben Sie sie mit print() aus:

  1. eine Liste mit den Schraubenschlüssel-Größen 10, 13, 17, 19, 22 (gemessen in mm),

  2. eine Liste mit den Materialnamen “Stahl”, “Aluminium”, “Titan”, “Kupfer” und

  3. eine Liste mit gemischten Datentypen für ein Bauteil: Name “Welle”, Material “Stahl”, Gewicht 12.5, Anzahl 3.

Frage: Welche Datentypen enthält die dritte Liste?

# Code-Zelle
Lösung
# Liste 1: Schraubenschlüssel-Größen
schraubenschluessel = [10, 13, 17, 19, 22]
print(schraubenschluessel)

# Liste 2: Materialnamen
materialien = ["Stahl", "Aluminium", "Titan", "Kupfer"]
print(materialien)

# Liste 3: gemischte Datentypen für ein Bauteil
bauteil = ["Welle", "Stahl", 12.5, 3]
print(bauteil)

Antwort zur Frage: Die dritte Liste enthält vier Elemente mit folgenden Datentypen:

  • Index 0: String (“Welle”)

  • Index 1: String (“Stahl”)

  • Index 2: Float (12.5)

  • Index 3: Integer (3)

Übung 3.2

Betrachten Sie folgende Liste mit Temperaturmessungen:

temperaturen = [20.5, 22.3, 21.8, 23.1, 24.5]

  1. Sagen Sie vorher, was die folgenden Anweisungen ausgeben:

    • print(temperaturen[0])

    • print(temperaturen[2])

    • print(temperaturen[4])

  2. Führen Sie den Code in der Code-Zelle aus und überprüfen Sie Ihre Vorhersage.

  3. Erklären Sie: Warum gibt temperaturen[2] die dritte Temperatur aus und nicht die zweite?

# Code-Zelle
Lösung

  1. Vorhersage:

    • temperaturen[0] gibt 20.5 aus (erstes Element)

    • temperaturen[2] gibt 21.8 aus (drittes Element)

    • temperaturen[4] gibt 24.5 aus (fünftes Element)

  2. Code zur Überprüfung:

temperaturen = [20.5, 22.3, 21.8, 23.1, 24.5]
print(temperaturen[0])
print(temperaturen[2])
print(temperaturen[4])

  1. Erklärung: Python verwendet eine 0-basierte Indexierung. Das bedeutet:

    • Index 0 = 1. Element (Position 1)

    • Index 1 = 2. Element (Position 2)

    • Index 2 = 3. Element (Position 3)

    • und so weiter...

Der Index gibt also nicht die Position an, sondern zählt ab 0. Daher liefert temperaturen[2] das dritte Element, nicht das zweite!

Übung 3.3

In einem Werkzeugkoffer wurde ein Schraubenschlüssel falsch einsortiert. Gegeben ist folgende Liste:

werkzeugkoffer = ['8 mm', '10 mm', '17 mm', '19 mm', '24 mm']

Die Größe an Position 3 (Index 2) ist falsch. Es sollte ein 13-mm-Schraubenschlüssel sein.

  1. Korrigieren Sie die Liste, indem Sie das falsche Element durch ‘13 mm’ ersetzen.

  2. Geben Sie die korrigierte Liste aus, um das Ergebnis zu überprüfen.

# Code-Zelle
Lösung
# gegeben: Liste mit Fehler
werkzeugkoffer = ['8 mm', '10 mm', '17 mm', '19 mm', '24 mm']

# Korrektur: ersetze Element an Position 3 (Index 2)
werkzeugkoffer[2] = '13 mm'

# Ausgabe zur Kontrolle
print(werkzeugkoffer)

Ausgabe:

['8 mm', '10 mm', '13 mm', '19 mm', '24 mm']

Erklärung: Mit dem Schreibzugriff werkzeugkoffer[2] = '13 mm' wird das Element an Index 2 (Position 3) durch den neuen Wert ersetzt. Der alte Wert ‘17 mm’ wird überschrieben.

Übung 3.5

Sie möchten eine Liste mit Drehzahl-Messungen (gemessen in Umdrehungen pro Minute) erstellen und diese schrittweise erweitern.

  1. Beginnen Sie mit einer leeren Liste drehzahlen_umin = [].

  2. Fügen Sie nacheinander folgende Drehzahl-Werte mit der Methode .append() hinzu: 500, 1000, 1500, 2000, 2500 (alle in U/min).

  3. Geben Sie nach jedem Hinzufügen die aktuelle Liste aus, um die Veränderung zu sehen.

  4. Bestimmen Sie am Ende mit len() die Anzahl der Messungen.

# Code-Zelle
Lösung
# Start mit leerer Liste
drehzahlen_umin = []

# Schrittweises Hinzufügen mit .append()
drehzahlen_umin.append(500)
print(drehzahlen_umin)

drehzahlen_umin.append(1000)
print(drehzahlen_umin)

drehzahlen_umin.append(1500)
print(drehzahlen_umin)

drehzahlen_umin.append(2000)
print(drehzahlen_umin)

drehzahlen_umin.append(2500)
print(drehzahlen_umin)

# Anzahl bestimmen
anzahl = len(drehzahlen_umin)
print('Anzahl der Messungen:', anzahl)

Ausgabe:

[500]
[500, 1000]
[500, 1000, 1500]
[500, 1000, 1500, 2000]
[500, 1000, 1500, 2000, 2500]
Anzahl der Messungen: 5

Erklärung: Die Methode .append() fügt ein Element am Ende der Liste hinzu. Nach jedem Aufruf wird die Liste um ein Element länger. Dies ist besonders nützlich, wenn Daten schrittweise erfasst werden (z.B. bei mehreren Messungen nacheinander).

Übung 3.6

In einem Lager wurden Bauteile gezählt. Die Liste enthält jedoch einen doppelten Eintrag, der korrigiert werden muss:

bauteile = ['Schraube M8', 'Mutter M8', 'Scheibe', 'Schraube M8', 'Unterlegscheibe']

Die Schraube M8 wurde versehentlich doppelt erfasst.

  1. Entfernen Sie die erste vorkommende ‘Schraube M8’.

  2. Geben Sie die bereinigte Liste aus.

  3. Entfernen Sie auch die zweite ‘Schraube M8’.

  4. Versuchen Sie anschließend nochmal ‘Schraube M8’ zu entfernen. Was passiert? Warum?

# Code-Zelle
Lösung
# gegeben: Liste mit doppeltem Eintrag
bauteile = ['Schraube M8', 'Mutter M8', 'Scheibe', 'Schraube M8', 'Unterlegscheibe']

# Entfernung des ersten Vorkommens
bauteile.remove('Schraube M8')
print(bauteile)

# Nochmaliges Entfernen
bauteile.remove('Schraube M8')
print(bauteile)

# Versuch, ein drittes Mal zu entfernen: erzeugt Fehler!
# bauteile.remove('Schraube M8')  # ValueError!

Ausgabe:

['Mutter M8', 'Scheibe', 'Schraube M8', 'Unterlegscheibe']
['Mutter M8', 'Scheibe', 'Unterlegscheibe']

Erklärung:

  • .remove() entfernt immer nur das erste Vorkommen eines Elements.

  • Nach dem ersten .remove() ist noch eine ‘Schraube M8’ in der Liste (an Position 4).

  • Nach dem zweiten .remove() gibt es keine ‘Schraube M8’ mehr.

  • Ein dritter Versuch würde einen ValueError erzeugen, weil das Element nicht mehr in der Liste existiert.

Übung 3.7

Sie planen eine WG-Party und erstellen gemeinsam mit Ihren Mitbewohnern eine Einkaufsliste. Gegeben sind zwei separate Listen:

einkauf_alice = ['Chips', 'Cola', 'Salsa', 'Nachos']
einkauf_bob = ['Pizza', 'Bier', 'Eis']

Bearbeiten Sie folgende Schritte:

  1. Verketten Sie beide Listen zu einer gemeinsamen Liste einkaufsliste.

  2. Alice fällt ein, dass noch Käse fehlt. Fügen Sie ‘Käse’ mit .append() hinzu.

  3. Bob ist spontan zum Veganer geworden und möchte keine Pizza mehr. Entfernen Sie ‘Pizza’ mit .remove().

  4. Alice möchte doch lieber Wasser statt Cola. Ersetzen Sie ‘Cola’ an der entsprechenden Position durch ‘Wasser’.

  5. Bestimmen Sie mit len() die finale Anzahl der Artikel.

  6. Geben Sie die finale Einkaufsliste und die Anzahl aus.

# Code-Zelle
Lösung
# gegeben: zwei separate Einkaufslisten
einkauf_alice = ['Chips', 'Cola', 'Salsa', 'Nachos']
einkauf_bob = ['Pizza', 'Bier', 'Eis']

# 1. Verkettung
einkaufsliste = einkauf_alice + einkauf_bob
print('Nach Verkettung:', einkaufsliste)

# 2. Käse hinzufügen
einkaufsliste.append('Käse')
print('Nach append:', einkaufsliste)

# 3. Pizza entfernen
einkaufsliste.remove('Pizza')
print('Nach remove:', einkaufsliste)

# 4. Cola durch Wasser ersetzen (Cola ist an Index 1)
einkaufsliste[1] = 'Wasser'
print('Nach Schreibzugriff:', einkaufsliste)

# 5. Anzahl bestimmen
anzahl = len(einkaufsliste)

# 6. Finale Ausgabe
print('Finale Einkaufsliste:', einkaufsliste)
print('Anzahl der Artikel:', anzahl)

Ausgabe:

Nach Verkettung: ['Chips', 'Cola', 'Salsa', 'Nachos', 'Pizza', 'Bier', 'Eis']
Nach append: ['Chips', 'Cola', 'Salsa', 'Nachos', 'Pizza', 'Bier', 'Eis', 'Käse']
Nach remove: ['Chips', 'Cola', 'Salsa', 'Nachos', 'Bier', 'Eis', 'Käse']
Nach Schreibzugriff: ['Chips', 'Wasser', 'Salsa', 'Nachos', 'Bier', 'Eis', 'Käse']
Finale Einkaufsliste: ['Chips', 'Wasser', 'Salsa', 'Nachos', 'Bier', 'Eis', 'Käse']
Anzahl der Artikel: 7

Übung 3.10 (Mini-Projekt)

Sie arbeiten in der Qualitätskontrolle und prüfen gefertigte Wellen. Der Soll-Durchmesser beträgt 50.0 mm. Bei 5 Bauteilen wurden folgende Ist-Durchmesser gemessen:

ist_durchmesser_mm = [50.2, 49.8, 73.5, 50.1, 49.9]

Ein Messwert ist offensichtlich falsch (Ausreißer durch Messfehler).

Das Programm soll folgende Aufgaben erfüllen:

  1. Fehleranalyse:

    • Geben Sie die Liste aus: “Gemessene Durchmesser: {Liste}”

    • Identifizieren Sie den unrealistischen Wert (Ausreißer).

  2. Datenbereinigung:

    • Entfernen Sie den Ausreißer mit .remove().

    • Geben Sie aus: “Bereinigte Durchmesser: {Liste}”

  3. Abweichungsberechnung:

    • Definieren Sie soll_durchmesser_mm = 50.0

    • Erstellen Sie eine leere Liste abweichungen_mm = []

    • Berechnen Sie für jeden bereinigten Wert die Abweichung: ist - soll

    • Fügen Sie jede Abweichung mit .append() zur Liste hinzu.

    • Hinweis: Sie benötigen 4 Berechnungen mit Index [0], [1], [2], [3]

  4. Auswertung:

    • Bestimmen Sie die Anzahl der gültigen Messungen mit len()

    • Geben Sie aus: “Anzahl gültiger Messungen: {Anzahl}”

    • Geben Sie die Abweichungen aus: “Abweichungen vom Soll: {Liste} mm”

  5. Visualisierung:

    • Erstellen Sie ein Liniendiagramm mit Plotly Express

    • x-Achse: Bauteilnummern [1, 2, 3, 4]

    • y-Achse: Abweichungen in mm

    • Titel: “Maßabweichungen der Wellen”

    • Achsenbeschriftungen: x = “Bauteil Nr.”, y = “Abweichung in mm”

Anforderungen:

  • Strukturieren Sie mit EVA-Kommentaren (Eingabe, Verarbeitung, Ausgabe)

  • Verwenden Sie aussagekräftige Variablennamen

# Code-Zelle
Lösung
# Import
import plotly.express as px

# Eingabe
ist_durchmesser_mm = [50.2, 49.8, 73.5, 50.1, 49.9]
soll_durchmesser_mm = 50.0

# Ausgabe der Rohdaten
print('Gemessene Durchmesser:', ist_durchmesser_mm)

# Verarbeitung: Datenbereinigung
# Der Ausreißer 73.5 ist offensichtlich unrealistisch.
ist_durchmesser_mm.remove(73.5)
print('Bereinigte Durchmesser:', ist_durchmesser_mm)

# Verarbeitung: Abweichungsberechnung
abweichungen_mm = []

abweichung_1 = ist_durchmesser_mm[0] - soll_durchmesser_mm
abweichungen_mm.append(abweichung_1)

abweichung_2 = ist_durchmesser_mm[1] - soll_durchmesser_mm
abweichungen_mm.append(abweichung_2)

abweichung_3 = ist_durchmesser_mm[2] - soll_durchmesser_mm
abweichungen_mm.append(abweichung_3)

abweichung_4 = ist_durchmesser_mm[3] - soll_durchmesser_mm
abweichungen_mm.append(abweichung_4)

# Auswertung
anzahl = len(ist_durchmesser_mm)
print('Anzahl gültiger Messungen:', anzahl)
print('Abweichungen vom Soll:', abweichungen_mm, 'mm')

# Ausgabe: Visualisierung
bauteilnummern = [1, 2, 3, 4]

fig = px.line(x=bauteilnummern, y=abweichungen_mm,
              title='Maßabweichungen der Wellen',
              labels={'x': 'Bauteil Nr.', 'y': 'Abweichung in mm'})
fig.show()

Beispiel-Ausgabe:

Gemessene Durchmesser: [50.2, 49.8, 73.5, 50.1, 49.9]
Bereinigte Durchmesser: [50.2, 49.8, 50.1, 49.9]
Anzahl gültiger Messungen: 4
Abweichungen vom Soll: [0.2, -0.2, 0.1, -0.1] mm

Anschließend wird das Liniendiagramm mit den Abweichungen angezeigt.

Erklärung: Dieses Mini-Projekt zeigt einen typischen Qualitätskontroll-Workflow:

  1. Messdaten werden analysiert (visuelle Prüfung)

  2. Ausreißer werden identifiziert und entfernt (remove)

  3. Abweichungen vom Sollwert werden berechnet (Subtraktion + append)

  4. Ergebnisse werden dokumentiert (len, print)

  5. Abweichungen werden visualisiert (Lineplot)

Positive Abweichungen bedeuten: Bauteil ist zu dick Negative Abweichungen bedeuten: Bauteil ist zu dünn

Das Diagramm zeigt, dass alle bereinigten Bauteile innerhalb ±0.2 mm vom Sollwert liegen.
